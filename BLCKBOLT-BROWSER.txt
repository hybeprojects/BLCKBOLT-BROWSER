PROJECT: BLCKBOLT-BROWSER  
TYPE: Advanced Developer Privacy Browser  
TECH: Electron + Node.js + JavaScript (extendable to TypeScript later)

GOAL:
Build a cross-platform desktop browser optimized for developer use and privacy protection.  
Must support modular extensions (VPN routing, ad/tracker blocking, fingerprint customization, and dev tools integration).

––––––––––––––––––––––––––
CORE REQUIREMENTS
––––––––––––––––––––––––––

1. *Project Setup*
   • Initialize a new Electron app with npm.
   • Create a secure main process (no nodeIntegration in renderer).
   • Structure folders as:
     /src
       /main
       /renderer
       /modules
       /assets
     /configs
   • Add build scripts in package.json: start, build, pack.

2. *UI*
   • Clean developer-style interface:  
     – URL bar with navigation (Back / Forward / Reload)  
     – Central webview area (BrowserView)  
     – Left sidebar with toggles: VPN, Proxy, AdBlock, Fingerprint, DevTools  
     – Bottom status bar for connection and routing info
   • Dark theme by default.  
   • Use Tailwind CSS for styling.

3. *Network + VPN Integration*
   • Module: /modules/network/vpn.js
     – Accept OpenVPN .ovpn configuration files.
     – Allow user to connect/disconnect via GUI toggle.
     – Display active server and connection status in status bar.
     – Use child_process to run OpenVPN CLI in background.
   • Optional: fallback to system proxy routing if OpenVPN not active.

4. *Ad + Tracker Blocking*
   • Module: /modules/adblocker.js
     – Implement filter rules using EasyList and EasyPrivacy.
     – Intercept requests through Electron session.webRequest.
     – Log number of blocked requests per site.

5. *Custom Fingerprint Control*
   • Module: /modules/fingerprint.js
     – Override user agent, screen size, and timezone on per-profile basis.
     – Option to randomize fingerprints at startup or per-tab.
     – Use navigator and window overrides via preload script.
   • Include UI toggle to view and refresh current fingerprint profile.

6. *Developer Features*
   • Integrate Electron DevTools by default (Ctrl+Shift+I).
   • Console tab in sidebar for quick logs.
   • Built-in JSON viewer and response inspector (later phase).

7. *Privacy + Security*
   • Disable telemetry.
   • Use isolated storage per profile directory.
   • Clear cookies/cache on exit (optional setting).
   • Implement auto-update checks for dependencies.

8. *Config Files*
   • /configs/settings.json → global preferences
   • /configs/vpn/ → default OpenVPN configs
   • /configs/adlists/ → EasyList URLs

9. *Branding*
   • App name: BLCKBOLT-BROWSER
   • Window title: “BLCKBOLT BROWSER – Developer Mode”
   • Icon: lightning bolt inside a shield (placeholder SVG ok)

10. *Scripts*
   npm start → launches development mode
   npm run build → packages distributable
   npm run lint → code lint
